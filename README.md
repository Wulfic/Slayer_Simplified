# Slayer Navigation

A RuneLite plugin that extends the Slayer Assistant with Shortest Path navigation integration. Adds "Navigate" buttons to slayer task locations that draw paths using the Shortest Path plugin's PluginMessage API.

## Features
- Browse slayer tasks with location details
- One-click navigation to any slayer task location
- Integrates with the Shortest Path plugin for pathfinding

## Requirements
- [Shortest Path](https://github.com/Skretzo/shortest-path) plugin installed in RuneLite

## Building
```
./gradlew build
./gradlew run
```
